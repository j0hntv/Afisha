## Установка:
```
git clone https://github.com/j0hntv/Afisha.git
python3 -m venv env
. env/bin/activate
pip install -r requirements.txt
```

## Цели проекта

Код написан в учебных целях — это урок в курсе по Python и веб-разработке на сайте [Devman](https://dvmn.org).
